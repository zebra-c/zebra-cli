# zebra-cli

[![npm version](https://badge.fury.io/js/%40zebra-c%2Fzebra-cli.svg)](https://badge.fury.io/js/%40zebra-c%2Fzebra-cli)

## 安装

```sh
npm install -g @zebra-c/zebra-cli
zebra-cli create my-project
```

## 创建项目

```sh
zebra-cli create my-project
```
